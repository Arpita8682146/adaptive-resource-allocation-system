# Node.js Frontend Scaffold

This folder now includes a basic `package.json` for future Node.js-based UI development.

## Next steps

1. Run `npm install` inside `src/web_app`.
2. Add a frontend entrypoint, such as `frontend/index.js` or a custom Streamlit component.
3. Build the UI and integrate it with Streamlit using `streamlit.components.v1`.

## Notes

- `streamlit-component-lib` is included so custom Streamlit components can be developed.
- `vite` is included as a lightweight build tool for future React/JS apps.
